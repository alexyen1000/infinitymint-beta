Interface InfinityMintProject

Interface

InfinityMint Project settings {@param name} - Name of the rpoejct {@param infinityLinks} - Specify custom InfinityLinks {@param system} - Specify whether the project is part of a grouped system with other projects. {@param modules} - Specify which InfintyMint modules will be used within the creation of tokens, such as RarityMinter, RoyaltyController. This is an immutable setting that cannot be changed or added with more. They are specific to the InfinityMint source code.
